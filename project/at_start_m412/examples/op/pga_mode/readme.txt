  this demo is based on the at-start board, in this demo, provides a description 
  of how to use op3 peripheral in programmable gain amplifier mode. user need to 
  generate a voltage waveform to the op3 input(pb0),op3 output(pb1) quadruple
  voltage waveform.
  note:input voltage must be less than 3.3v/4.
       operational amplifier must be calibrated before use.
