  this demo is based on the at-start board, in this demo, shows how to use
  the tmr peripheral input blanking function when #define ENABLE_BLANKING.
  set-up
  - connect the external signal (for example pa0) to the tmr3_ch2 pin (pa7) to measure.
  - connect tmr10_ch1 pin (pa12) to measure.
  - connect pin (pa2) to measure, for toggle pa2 when input capture of tmr3_ch2 occurs.
  
  for more detailed information. please refer to the application note document AN0236.
