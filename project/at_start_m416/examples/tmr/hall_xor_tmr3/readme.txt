  this demo is based on the at-start board, in this demo, pa8 output
  waveforms, pa8 toggled each pa3, pa6, pa7 edge.
  pin connection:
  - pb4    <--->    pa3
  - pb5    <--->    pa6
  - pb0    <--->    pa7 

  for more detailed information. please refer to the application note document AN0236.
